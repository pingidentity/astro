# `@pingux/astro`

Package of reusable UI components built on the top of [styled-system](https://styled-system.com/).

## Installing

Run `npm install @pingux/astro@beta` to get the latest beta version.

## Usage

Wrap your application's root into <AstroWrapper> to initialize context that will pass styling data:

```
<AstroWrapper>
  <App />
</AstroWrapper>
```
