# Box

Box is a flex-based component built to create rows and columns while controlling sizes and spacing. 
This component is based on the [Box-Theme UI](https://theme-ui.com/components/box/)
and accepts most [Styled System style props](https://styled-system.com/table/).

### Required components

- Box component can be used independently and does not require additional components. 

### Best practices

Box should:
- be used for styling form controls or as a container.

Box shouldnâ€™t:
- be used for large complex grid layouts.

### Accessibility

Use **`as`** prop to define container HTML semantics, like using `<article>` to identify blog posts. 
It complies with the [WCAG 2.1 - 1.3.1 Info and Relationships](https://www.w3.org/WAI/WCAG21/Understanding/info-and-relationships.html) 
accessibility guidelines.
