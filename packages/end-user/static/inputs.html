IE & Edge don’t support the CSS-only solution for float labels, but we can simulate the “placeholder-shown” pseudo-selector with a “placeholder-shown” class. (View in IE or Edge to see the difference.)

React 
<FloatLabelTextInput
  id="username5"
  inputClassName="placeholder-shown"
  label="Username"
 />
HTML 
<div class="float-label"><input type="text"
        class="text-input float-label__input placeholder-shown"
        data-id="text-input" id="username5" name="username5"
        placeholder="Username" /><label
        class="float-label__label"
        for="username5">Username</label></div>

 />

        name="dropdown"
