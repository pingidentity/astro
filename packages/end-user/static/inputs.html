IE & Edge don’t support the CSS-only solution for float labels, but we can simulate the “placeholder-shown” pseudo-selector with a “placeholder-shown” class. (View in IE or Edge to see the difference.)

React
<FloatLabelTextInput
  id="username5"
  inputClassName="placeholder-shown"
  label="Username"
 />
HTML
<div class="float-label">
    <input type="text" class="text-input float-label__input placeholder-shown"
        id="username5" name="username5" placeholder="Username" />
    <label class="float-label__label" for="username5">Username</label>
</div>

Stateful demo

This component toggles the “placeholder-shown” class depending on whether there’s a value.

React
<withState(mapProps(withHandlers(FloatLabelDropdown)))
  label="Select one"
  options={["First Value","Second Value","Third Value","Fourth Value"]}
 />
HTML
<div class="dropdown float-label">
    <select id="dropdown" name="dropdown" class="dropdown__select float-label__input placeholder-shown">
        <option selected="" disabled="" value="">Select one</option>
        <option value="First Value">First Value</option>
        <option value="Second Value">Second Value</option>
        <option value="Third Value">Third Value</option>
        <option value="Fourth Value">Fourth Value</option>
    </select>
    <label class="float-label__label" for="dropdown">Select one</label>
</div>
